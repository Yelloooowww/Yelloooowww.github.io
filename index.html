Projects & Experiences

Human Robot
Control System


This is our graduation project, guided by professor Jiang-Shyh-Biau at MVMC Lab, NCUME.
This project replaced the core of KONDO KHR-3HV with our microcontroller, ASA_M128.
We designed a control system and built the power, communication and data storage systems,
implemented the system in a C program and designed a GUI in Python.

Color Tracking
Car


This color-tracking car was designed with an Raspberrypi 3B+, a webcam and the DC motor driving system.
The car can move automatically to track the object with the color be chosen.
This project used Camshift algorithm to track the motion object and programmed in C++ with Opencv library.

Automatic Mobile
Car


During spring semester 2019, I taught in CLHS as a teaching assistant for an elective course.
I taught students how to design a Automatic Car with ASA_M128, some infrared sensors and the stepping motor driving system.
The car could finish the maze in the limited time.
